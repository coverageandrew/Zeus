## Intake Summary

**Paw Safe** is a community-driven pet safety and travel companion app.

### Core Features (MVP - Phase 1)
1. **Community Hub** — Location-based pet discussions, alerts, lost pet notifications
2. **Safety Alerts** — Real-time hazard reports with credibility markers
3. **Lost Pet Network** — Geo-tagged alerts, community search, safe contact
4. **Smart Route Planning** — Google Maps integration, pet-friendly POI discovery
5. **Pet Profiles** — Breed, age, personality, photos
6. **Premium Subscriptions** — Advanced features via Stripe

### Tech Stack
- **Frontend:** React (Vite) + TypeScript + Tailwind CSS + shadcn/ui
- **State:** TanStack Query + Zustand
- **Database:** Supabase (Postgres + PostGIS)
- **Auth:** Supabase Auth (email/password + Google OAuth)
- **Maps:** Google Maps API
- **Payments:** Stripe
- **Deployment:** Vercel

### Key Differentiators
- Neighborhood-first model (not follower-based)
- Lightweight trust/verification for anti-scam
- Emergency Mode for lost pets
- Pet-friendly routing optimized for travelers
